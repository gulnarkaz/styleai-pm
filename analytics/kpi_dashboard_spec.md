# StyleGenie AI: KPI Dashboard Specification

**Цель:** Обеспечение прозрачности продукта для стейкхолдеров и оперативный мониторинг здоровья AI-системы.

---

### 1. Business Metrics (Бизнес-показатели)
*Эти метрики показывают финансовую эффективность StyleGenie для ритейл-партнеров.*

| Метрика | Описание | Целевое значение |
| :--- | :--- | :--- |
| **Conversion Rate (CR)** | Процент пользователей, совершивших покупку после совета ИИ. | **2.2%** |
| **Average Order Value (AOV)** | Средний чек пользователя (сравнение с пользователями без ИИ). | **+10%** |
| **Return Rate (RR)** | Процент возвратов товаров, купленных по рекомендации ИИ. | **< 15%** |
| **LTV Contribution** | Рост пожизненной ценности клиента за счет удержания в экосистеме. | **+12%** |

---

### 2. AI Product Quality (Качество продукта)
*Метрики "здоровья" модели и удовлетворенности пользователей.*

* **CSAT (Customer Satisfaction):** Оценка пользователем качества совета (1-5 звезд). **Target: > 4.2**.
* **Discovery Rate:** Доля "Wildcard" рекомендаций, принятых пользователем. **Target: 15%**.
* **Session Completion:** Процент диалогов, закончившихся переходом в корзину.
* **Latency:** Скорость ответа модели. **Target: < 2.5 сек**.

---

### 3. Ethics & Governance (Этика и комплаенс)
*Уникальный блок метрик для мониторинга Bias и безопасности.*

* **Fairness Gap:** Разница в CSAT между сегментами (Plus-size / Standard / Petite). **Threshold: < 15%**.
* **Hallucination Rate:** Доля ответов, содержащих ошибки в характеристиках товара (на основе ручного аудита). **Target: < 2%**.
* **Guardrail Interventions:** Частота срабатывания фильтров безопасности на токсичные или некорректные запросы.
* **Privacy Compliance:** Время жизни исходных фото на сервере (мониторинг удаления).

---

### 4. Визуализация (Dashboard Layout)

#### **Блок 1: Real-time Health** (Верхняя панель)
* Светофор статуса: **Green** (Fairness < 15%), **Yellow**, **Red**.
* Текущий CR vs. Target (линейный график).

#### **Блок 2: Funnel Analysis** (Воронка)
1. Фото загружено -> 2. Анализ силуэта -> 3. Рекомендация выдана -> 4. Переход в корзину -> 5. Покупка.

#### **Блок 3: Segment Performance** (Круговые диаграммы)
* Распределение рекомендаций по типам фигур и цветотипам для контроля репрезентативности выборки.
